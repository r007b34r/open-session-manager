# Query API Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 补齐 `API-06`，让 `list/search/get/view/expand` 不只存在于 CLI，也能作为 Tauri command 对外暴露，明确与 CLI 复用同一套 Rust 查询层。

**Architecture:** 不再让桌面端只暴露整个 dashboard snapshot。新增五个 Tauri command 包装器，内部统一复用 `src-tauri/src/commands/query.rs` 的查询函数。测试层用临时 HOME + fixture 目录直接调用这些 async commands，验证结果结构与 CLI 一致，而不是再复制一份查询逻辑。

**Tech Stack:** Rust, Tauri command, cargo test
