# OpenAPI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 补齐 `API-03`，为现有本地 REST API 生成并暴露可验证的 OpenAPI 文档。

**Architecture:** 继续复用 `api_server` 里的同一套路由，不引入额外框架层。新增 `openapi` 模块输出稳定的 OpenAPI JSON 文档，服务端通过 `/openapi.json` 直接暴露，测试同时验证 schema 基本结构和示例字段。

**Tech Stack:** Rust, serde_json, axum, cargo test
